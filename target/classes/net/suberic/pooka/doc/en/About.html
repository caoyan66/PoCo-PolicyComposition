Pooka normally ships with the following support libraries:

  • JavaBeans(TM) Activation Framework version 1.0.2 by Sun Microsystems
  • JavaMail(TM) version 1.3.3_01 by Sun Microsystems
  • JavaHelp(TM) version 1.1.3 by Sun Micorsystems
  • Knife mbox provider (from http://dog.net.uk/knife
  • Kunststoff Pluggable Look and Feel from incors.org
  • JavaMail Maildir provider from Alexander Zhukov
  • JDesktop Integration Components from https://jdic.dev.java.net/

For encryption, Pooka uses

  • Bouncy Castle Cryptography JCE and S/MIME provider (http://www.bouncycastle.org/)
  • Cryptix JCE and OpenPGP provider (http://www.cryptix.org/)
  • JavaMail-Crypto API (http://javamail-crypto.sourceforge.net/)
