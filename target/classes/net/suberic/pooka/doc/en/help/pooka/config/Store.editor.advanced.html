Incoming Mail Accounts - Advanced Settings

This screen gives you a list of the Mail Accounts that you have configured in Pooka. Select "Add" to create a new Mail Account, "Edit" to edit an existing Mail Account, or "Remove" to remove a Mail Account.

Store debug
Sets the debug log level for Pooka for this mail Store. Debug messages are printed to the console.
Mail Session debug
Sets the debug log level for the JavaMail session for this mail store. Debug messages are printed to the console.
Connection
Sets the connection to use for this store. See Network Connections for more detail.
