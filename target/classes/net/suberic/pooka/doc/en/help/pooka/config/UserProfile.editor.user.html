Email Profiles - User

Email Address
The email address for this Profile. This will be used for the From: field.
Your Name
The personal name for this address.
Reply-To Address
The email address that will be the default Reply-To for this Profile. If blank, then by default replies will go to the main Email Address.
Reply-To Name
The personal name for the reply-to.
Organization
The Organization to use.
Append Signature to Each Message
Whether or not the signature file is automatically appended to each message. If unset, then you can still manually add signatures to messages.
Signature File
A text file that will be used as a signature.
Save Messages to Sent Folder
Check this if you want all messages sent using this Profile to be saved to a particular Sent Folder.
Sent Folder
The Sent Folder to which outgoing messages using this Profile will be saved.
SMTP Server
The SMTP Server to use by default for this Profile.
